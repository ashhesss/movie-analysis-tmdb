# Анализ фильмов: что влияет на сборы и рейтинг

Итоговый проект по дисциплине **«Анализ данных на Python»**.
Исследование на данных [The Movie Database (TMDB)](https://www.themoviedb.org/):
сбор через API, очистка, проверка распределений и гипотез, регрессия и визуализация.

## Команда

- Егоров Глеб — сбор и очистка данных
- Безматерных Всеволод — разведочный анализ и визуализация
- Филатов Евгений — статистические тесты и регрессия

## Исследовательские вопросы

1. Связан ли бюджет фильма с его кассовыми сборами?
2. Влияет ли хронометраж на рейтинг?
3. Различается ли средний рейтинг между жанрами?

## Данные

- **Источник:** TMDB API, эндпоинты `/discover/movie` и `/movie/{id}`.
- **Объём:** 600 фильмов, отсортированных по числу оценок (репрезентативная
  выборка популярных фильмов).
- **Способ сбора:** пагинация по страницам выдачи + дозапрос деталей по каждому
  фильму (бюджет, сборы, хронометраж, жанры).
- **Признаки:** `budget`, `revenue`, `runtime`, `vote_average`, `vote_count`,
  `popularity`, `genres` и расчётные — `roi`, `profit`, `decade`, `main_genre`.

## Как запустить

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/<ваш-логин>/movie-analysis-tmdb.git
   cd movie-analysis-tmdb
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Получите бесплатный ключ TMDB: [themoviedb.org](https://www.themoviedb.org/)
   → Settings → API → ключ **v3 auth** (короткая строка из 32 символов).
4. Откройте `notebooks/movies_analysis_tmdb.ipynb` в Jupyter или
   [Google Colab](https://colab.research.google.com), введите ключ при запуске и
   выполните ячейки сверху вниз.

> **Важно:** ключ вводится при запуске и не хранится в репозитории.

## Ход работы

1. **Сбор данных** — пагинация по TMDB API + дозапрос деталей.
2. **Очистка** — нули в финансовых полях → пропуски, заполнение медианой,
   новые признаки.
3. **Разведочный анализ** — статистика по жанрам и десятилетиям,
   корреляционная матрица, матрица диаграмм рассеивания.
4. **Проверка нормальности** — тест Шапиро-Уилка, асимметрия и эксцесс.
5. **Проверка гипотез** — корреляции Пирсона и Спирмена, F-критерий (ANOVA).
6. **Регрессия** — простая и множественная линейная регрессия.

## Основные результаты

- **Бюджет ↔ сборы:** корреляция Пирсона r = 0.69 (p < 0.001, n = 594) —
  умеренно-сильная положительная связь; R² ≈ 0.48.
- **Хронометраж ↔ рейтинг:** Спирмен rho = 0.21 (p < 0.001) — связь значима,
  но слабая.
- **Рейтинг по жанрам:** F-критерий F = 4.19, p = 0.003 — различия значимы;
  выше всех драма (7.79) и криминал (7.68), по кассе лидируют семейные и
  приключенческие фильмы.
- **Регрессия:** простая модель «бюджет → сборы» — R² = 0.58, каждый доллар
  бюджета приносит ≈ 3.55 $ сборов; в множественной модели значимы бюджет и
  число голосов, хронометраж — нет.
- **Нормальность:** распределения бюджета, сборов и хронометража скошены вправо
  и не проходят тест Шапиро-Уилка, поэтому для связей используется Спирмен.
